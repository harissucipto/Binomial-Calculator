Masukan p(a)adalah peluang muncul sampel

Masukan p(s)adalah Ruang Sampel

P(X) = ( n! / (n - x)! x! ) * p^x * q^(n-x)

kesimpulan rumus:
  • n = banyak percobaan
  • x = banyak berhasil
  • p = peluang berhasil
  • q = peluang gagal (q = 1 -p)
